Training and Validation

This page describes how the dataset is divided

The dataset is subjected to preprocessing stage where the entire dataset is divided into two parts:

  • Training set
  • Validation set


    • The dataset is divided into 80% of training set and 20% of Validation set to get the maximum accuracy in predicting the national flags.The model is built using tflearn. Returns accuracy of 0.525 on the training set and 0.66 on the validation set.
